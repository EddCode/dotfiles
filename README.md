# Dotfiles

> Create a default osx setting

* Fish shell
* Fisher
* Hombrew
* [Tmux](https://github.com/tmux/tmux/wiki)
* Vim plugins

## Hombrew
`brew.sh` file install homebrew if isnt installed and install some utiliies and cask brew as
`caskroom/cask` and `homebrew/cask-fonts`
 
*Homebrew formulas*
* GIT
* [Tmux](https://github.com/tmux/tmux/wiki)
* Nodejs
* *homebrew/cask-fonts*
  * font-firaCode-nerd-fon [from Nerd-Font](https://github.com/ryanoasis/nerd-fonts#font-installation)
* *caskroom/cask*
  * [Docker](https://www.docker.com)
  * [lastpass]()
  * [microsoft-edge](https://www.microsoft.com/en-us/edge) Web browser
  * [Postman](https://www.postman.com/product/api-client/)
  * Telegram
  * Slack
  * Whatsapp
  
## Fish
Set a fish shell as a default shell and install *[fisher](https://github.com/jorgebucaran/fisher)* to handle `nvm`
and set a vim as default editor

## TMUX
Change a key bind action to `^-z`

## Vim
Download repository wich has a some vimrc configs and a some plugins 
> to see wich plugins and configuration has it, take a look [EddCode/vimrc](https://github.com/EddCode/vimrc)
